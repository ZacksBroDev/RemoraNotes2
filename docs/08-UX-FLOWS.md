# RemoraNotes - UX Flows & Frontend Design

## Design Principles

1. **Mobile-first**: Designed for thumb navigation on 375px+ screens
2. **Neutral palette + single accent**: Gray/slate base with blue (#3B82F6) accent
3. **Large touch targets**: Minimum 44Ã—44px tap areas
4. **Clear hierarchy**: Typography drives visual structure
5. **No clutter**: White space is a feature, not wasted space
6. **No dark patterns**: Honest UX, clear actions, easy to undo
7. **Accessibility**: WCAG 2.1 AA compliant (contrast, focus states, labels)
